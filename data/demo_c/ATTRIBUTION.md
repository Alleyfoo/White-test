# Mushroom Commons Images — Set C: poor-quality / awkward-photo stress set

This folder contains five Wikimedia Commons mushroom images selected for a poorer, less textbook-clean stress-test set. The purpose is to test how model behaviour changes when the photo is blurry, low-resolution, cropped, cluttered, or otherwise less ideal.

These images are for benchmark/demo purposes only. They are not mushroom identification guidance.
